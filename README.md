## Usage

Build with the command

```sh
mvn package
```

This will generate sources, compile, run all unit tests, and produce a JAR file named `target/lambda-VERSION.jar`.

Run the graphical interface with

```sh
./gui
```

or the simple REPL with

```sh
./repl
```
